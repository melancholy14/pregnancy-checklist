# Contact 페이지 톤 변경

> 작성일: 2026-04-09 | 작성자: Claude Code

## 개요

Contact 페이지의 기업 고객센터 톤("연락처", "소중한 의견")을
1인 개발자 당사자 톤("의견 보내기", "혼자 만들다 보니 놓치는 것도 많아요")으로 전환합니다.

---

## 구현 내용

### 완료 조건 충족 여부

| 조건 | 상태 | 비고 |
|------|------|------|
| 타이틀 "연락처" → "의견 보내기" 변경 | ✅ 완료 | metadata 포함 |
| 기업 화법 → 1인 당사자 톤 변경 | ✅ 완료 | "소중한 의견" → "혼자 만들다 보니" |
| 피드백 폼 링크 유지 | ✅ 완료 | NEXT_PUBLIC_FEEDBACK_FORM_URL 조건부 렌더링 |
| 이메일 섹션 톤 변경 | ✅ 완료 | "문의사항이 있으시면" → "직접 연락 주셔도 돼요" |
| 이메일 mailto 링크 | ✅ 완료 | 텍스트 → 클릭 가능한 mailto 링크 |

### 생성/수정 파일

- `src/app/contact/page.tsx` — 타이틀, 카피, metadata, 레이아웃 전면 변경

### 주요 결정 사항

- **버튼 스타일 통일**: About 페이지와 동일하게 `bg-primary text-primary-foreground` 사용 (기존 하드코딩 `#6B5A80` 제거)
- **이메일을 mailto 링크로**: 리스트 아이템 → 클릭 가능한 링크로 UX 개선
- **prose 클래스 제거**: About 페이지와 동일하게 `max-w-lg mx-auto` 레이아웃으로 통일

### 가정 사항 및 미구현 항목

- PRD의 카피 문구를 그대로 사용
- 미구현 항목 없음
