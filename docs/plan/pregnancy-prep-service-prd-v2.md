# Pregnancy Preparation Service PRD

Version: 0.2\
Scope: MVP Web Service\
Stack: Next.js Fullstack + GCP-only

------------------------------------------------------------------------

# 1. Product Overview

임산부와 가족이 **출산 준비를 체계적으로 관리**할 수 있도록 돕는 서비스.

핵심 목적

-   출산 준비 체크리스트 제공
-   출산 예정일 기반 타임라인 자동 생성
-   연도별 베이비페어 정보 제공
-   임신 중 체중 기록 및 시각화
-   출산/육아 관련 영상 큐레이션

서비스 특징

-   회원가입 없이 사용 가능
-   출산 예정일 입력만으로 개인화 기능 제공
-   개인 데이터는 브라우저 LocalStorage 저장

------------------------------------------------------------------------

# 2. Goals

### Product Goals

1.  임산부가 **출산 준비 전체 흐름을 이해할 수 있도록 지원**
2.  검색 기반 트래픽 확보 (SEO)
3.  가입 없이 즉시 사용 가능한 UX 제공
4.  체크리스트 중심 준비 관리

------------------------------------------------------------------------

# 3. Permanent Non‑Goals

이 서비스는 앞으로도 다음 기능을 제공하지 않는다.

-   의료 상담 기능
-   의료 진단 또는 처방
-   개인 맞춤 의료 조언
-   병원 추천 기능
-   병원 예약 연동

본 서비스는 **정보 제공 및 준비 관리 도구**이다.

------------------------------------------------------------------------

# 4. MVP Scope

## Included

-   출산 예정일 입력
-   출산 준비 체크리스트
-   임신 주차 기반 타임라인
-   베이비페어 정보 제공
-   체중 기록
-   유튜브 영상 큐레이션

## Excluded

-   회원가입
-   남편 공유 기능
-   서버 기반 개인 데이터 저장
-   커뮤니티 기능
-   메시징 기능

------------------------------------------------------------------------

# 5. Core Features

## Due Date Input

사용자가 출산 예정일을 입력한다.

임신 시작일 계산

pregnancy_start = due_date - 280 days

현재 주차 계산

pregnancy_week = (today - pregnancy_start) / 7

------------------------------------------------------------------------

## Checklist

카테고리

-   병원 준비
-   출산 가방
-   신생아 준비
-   산후 준비
-   행정 준비

사용자는 항목을 체크할 수 있다.

체크 상태는 LocalStorage 저장.

------------------------------------------------------------------------

## Timeline

예

20주 → 베이비페어 방문\
28주 → 출산 가방 준비\
32주 → 카시트 준비
