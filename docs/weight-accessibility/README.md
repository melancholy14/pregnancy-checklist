# Phase 2.5 Step 7: 체중관리 접근성 개선

## 개요

홈 체중 카드를 미니 대시보드로 개선하고, 체중 페이지에 블로그 크로스 링크를 추가하여 체중관리 기능의 접근성과 콘텐츠 연결성을 높인다.

---

## 구현 내역

### 1. 홈 체중 미니 대시보드 개선

기록 건수별 3단계 상태 분기:

| 상태 | 카드 내용 | CTA |
|------|----------|-----|
| 0건 | "아직 기록이 없어요" | 시작하기 |
| 1건 | 최근 체중 (예: 62.5kg) | 추가 기록 |
| 2건+ | 최근 체중 + 변화량 (예: +2.1kg (처음 대비)) | 기록하기 |

### 2. 체중 페이지 블로그 연결

체중 기록 목록 하단에 `pregnancy-weight-management` 아티클 링크 카드 추가:
- 제목: "임신 중 체중 관리 가이드"
- 설명: "BMI별 권장 범위부터 안전한 운동법까지"
- 클릭 시 `/articles/pregnancy-weight-management`로 이동
